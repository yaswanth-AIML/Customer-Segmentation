# Customer Segmentation App

An interactive **Streamlit frontend** for customer segmentation using **unsupervised learning (KMeans clustering)**.  
Users can input customer details (age, income, purchases, visits, etc.) and the app predicts their cluster in real time.

## 🚀 Features
- Built with **Streamlit** for a clean UI
- Uses **KMeans clustering** for unsupervised segmentation
- Integrated **scaler + trained model** for accurate predictions
- Visualizations for insights (income vs. education, marital status vs. spending, etc.)

## 🛠️ Tech Stack
- Python
- Pandas
- Scikit-Learn
- Matplotlib
- Streamlit
